# 🧠 AI Resume Screening & Job Matching System

## 📌 Overview

This project is an intelligent resume screening system that matches resumes with job descriptions using Natural Language Processing (NLP) and Machine Learning techniques.

## 🚀 Features

* 📄 Upload PDF resumes
* 🧠 Semantic similarity matching (Sentence Transformers)
* 📊 Candidate ranking system
* 🏆 Top candidate identification
* 📈 Similarity visualization
* 📥 Downloadable reports (CSV/PDF)
* 🧩 Skill overlap analysis

## 🛠️ Technologies Used

* Python
* Gradio
* Sentence Transformers
* PDFMiner
* Pandas & Matplotlib

## ▶️ How to Run

```bash
pip install -r requirements.txt
python app.py
```

## 🌍 Deployment

The project can be deployed using Hugging Face Spaces or Heroku.

Developed as part of MSc Computer Science project focusing on AI-based recruitment automation.
